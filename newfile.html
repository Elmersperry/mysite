Основные семантические элементы на странице:

Шапка сайта - header (верхняя часть страницы)

Подвал сайта - footer (нижняя часть страницы)

Блок информации - article (целостный блок информации на странице)

Блок section - объединяет связанные между собой куски информации докумена (новости, объединенные по категории) Должен содержать заголовки h1-h6

aside - боковая панель main - основное, неповторяющееся содержимое страницы
